# 资源管理指南

## 资源类型

平台支持 8 种资源类型：

| 类型 | 标识 | 说明 | 示例 |
|------|------|------|------|
| 提示词 | prompt | AI 提示词模板，支持变量替换 | 代码审查助手、API 文档生成 |
| 技能 | skill | 可复用的技能/知识包 | React 组件开发指南、Prisma 最佳实践 |
| 组件 | component | UI 组件源码和文档 | Button、Modal、Table 等可复用组件 |
| MCP | mcp | Model Context Protocol 服务 | 文件系统、数据库连接 MCP |
| 协议 | protocol | 自定义 API 协议定义 | RESTful API 规范、GraphQL Schema |
| 工作流 | workflow | 可视化工作流模板 | CI/CD 流水线、数据迁移流程 |
| 模板 | template | 项目/代码模板 | React 后台模板、NestJS 服务模板 |
| 代码片段 | snippet | 可复用的代码片段 | 工具函数、自定义 Hooks |

## 创建资源

1. 点击侧边栏「新建资源」或使用快捷键 `Cmd/Ctrl + N`
2. 选择资源类型
3. 填写资源信息：
   - **名称**：清晰描述资源用途
   - **描述**：详细说明资源功能和适用场景
   - **标签**：便于搜索和分类
   - **可见性**：公开 / 团队 / 私有
4. 编辑内容（Markdown / JSON / YAML 格式）
5. 保存并发布

## 版本管理

- 每次修改可创建新版本（自动递增语义化版本号）
- 支持版本对比（Diff 视图高亮新增/删除/修改）
- 版本状态：草稿 → 审核 → 批准 → 发布
- 支持一键回滚至历史版本

## 搜索与筛选

- **全文搜索**：搜索名称、描述、标签
- **类型筛选**：按资源类型过滤
- **标签筛选**：按标签分类浏览
- **排序**：最新 / 最热 / 评分最高 / 下载最多

## 协作

- 支持团队共享（设置资源归属团队）
- 收藏功能（Star）
- 下载计数统计
